This is a simple Visual Studio 2019 C# program for storing accommodation
information for real-estate agents.
Project will store 1 photo for each Accommodation and can search for an
Accommodation based on 1 factor ( ID, Rooms, Size, etc )

Code and Algorithm
On the form 1 (main menu), the agent will choose either he/she wants to add property,
search or view. If there are no records saved (the file is not existing or it is empty) the
show property form will give a message and also everything on that form except the
exit button will be disabled (buttons, labels etc). This also applies for the “search for a
property” form.
If user chooses to add property, the form for adding will be shown and user
can enter the property information. If there's any wrong or empty data, the
program will indicate that there’s an error. Each property will have a unique ID
that is created automatically using letters and numbers.
Agents will be able to search depending on at least one element in the search
form (price, rooms, ID etc). In the show property form, users can travel
between all already saved records with ">>” (forward) and “<<” (backward)
buttons. The “>>” (forward) button will be disabled when the last record is
shown, and if the user comes back to previous record, the button will get
enabled again.

roll some more. He received four. He now has nine dollars in his account. The
banked points would be added to his total score if he presses the save button.

turn and all of their points.
