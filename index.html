Introduction
This is a simple Visual Studio 2019 C# dice game that can be enjoyed
by two to four players. The purpose of the game is to be the first to hit
the target, in order to win.
Example;
Target is = 20 | Tree players game
Player 1 rolls dice, saves & collects 20 points throughout gameplay
Player 1 wins the game.
Photos
Game settings menu before initiating the game and after the game is initiated.
With 4 players & 20 being the game goal.
Game play menu before the game has commenced and after.
Different turns for each players on the game to roll the dice and save to score.
Code and Algorithm
Initially the user will click play for the game settings menu to open.
The user then enters the amount of players that will play and set the game goal,
from 0 to 50 points.
The program will not let user enter more than 4 & less than 2 players, and not more
than 50 for game goal.
User then will enter the players name and choose their individual colors.
Only two groupboxes for player names will be available if the user chooses two
players.
Then the user must press start to begin the game. Each player's panels and labels
are divided, and they are colored according to the color that the consumer has
selected for that player. If the user selects red for player 1, the screen for that player
will be red in the game window. Additionally, two buttons, "save" and "roll," will be
red on player 1's turn.
All the that player rolls will go into his or her bank. The customer then has the option
of saving the number or rolling for more. If the customer earns 1, all of his or her
bank points will be forfeited. Player 1 rolled a 5 for example. He then proceeded to
roll some more. He received four. He now has nine dollars in his account. The
banked points would be added to his total score if he presses the save button.
But if user decides to roll once more time and rolls 1, as a result, user will miss their
turn and all of their points.
The game is won by the first person to cross the target.
